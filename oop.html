Object-Oriented Programming Concepts

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data and code: data in the form of fields (often known as attributes or properties), and code in the form of procedures (often known as methods).

Understanding OOP principles is essential for building scalable, maintainable, and reusable code. Our calculator application demonstrates these concepts through practical implementation.

Core OOP Principles Demonstrated

Encapsulation

Bundling data and methods that operate on that data within a single unit (class). Our Calculator class encapsulates all calculation logic.

Inheritance

Creating new classes from existing ones. Specialized calculators like XPYCalculator and UYTCalculator inherit from base Calculator.

Polymorphism

Methods performing differently based on the object. Different calculator types implement operations in their own way.

Abstraction

Hiding complex implementation details. Users interact with simple interfaces while complex logic stays hidden.

Code Example

// Base Calculator Class
class Calculator {
  constructor() {
    this.result = 0;
    this.history = [];
  }
  
  add(a, b) {
    const result = a + b;
    this.logOperation('add', [a, b], result);
    return result;
  }
  
  logOperation(op, inputs, output) {
    this.history.push({ op, inputs, output, time: Date.now() });
  }
}

// Specialized Calculator
class XPYCalculator extends Calculator {
  power(base, exponent) {
    const result = Math.pow(base, exponent);
    this.logOperation('power', [base, exponent], result);
    return result;
  }
}
