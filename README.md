# my-calculator
it is a web based calculator
this project consist of 1 HTML file, 1 styling file and 2 JS files 
vendor.js file connect the JS to HTML file 
app.js is the core js for my calculator

## Pages
- **index.html** - Home page with navigation to all calculators
- **yuyu.html** - YUYUYA Calculator page with specialized YUYUYA functionality
- **asd.html** - ASD Calculator page
- **xpy/index.html** - X Power Y (XPY) Calculator
- **uyt/index.html** - U Yield T (UYT) Calculator
- **basics-10-function-refactoring/index.html** - Basic Calculator with arithmetic operations
